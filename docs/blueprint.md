# **App Name**: MAW Soluciones

## Core Features:

- Presentación de Servicios: Muestra los servicios de la agencia con tarjetas modernas y visualmente atractivas. Los servicios incluyen creación de contenido, desarrollo web, producción de fotos/videos, automatización y gestión de campañas.
- Portafolio en Grid: Un grid de portafolio filtrable que muestra proyectos exitosos con imágenes grandes y descripciones breves.
- Formulario de Captura de Leads: Un formulario de contacto simple integrado con un botón directo de WhatsApp para capturar leads.
- Reserva de Sesión Estratégica: Botones de llamada a la acción prominentes en todo el sitio que enlazan a un sistema de programación para reservar sesiones estratégicas.
- Testimonios de Clientes: Muestra testimonios auténticos de clientes con fotos de perfil en un diseño limpio basado en tarjetas.
- Cronología de Logros: Muestra los hitos y logros clave de la agencia en un formato de cronología visualmente atractivo dentro de la sección 'Acerca de Nosotros'.

## Style Guidelines:

- Color primario: Negro profundo (#000000) para una sensación de sofisticación y profesionalismo.
- Color de fondo: Gris muy claro (#F0F0F0), casi blanco, para maximizar el enfoque en el contenido y permitir toques de color de otros elementos.
- Color de acento: Violeta Eléctrico (#8F00FF) para resaltar elementos interactivos clave y CTAs.
- Fuente del cuerpo: 'PT Sans', una sans-serif humanista, para una sensación moderna pero accesible.
- Fuente del título: 'Playfair', una serif moderna con alto contraste, para dar una sensación elegante y de alta gama. Use 'PT Sans' para el texto del cuerpo si se anticipa un texto más largo.
- Utilice un conjunto consistente de iconos minimalistas para representar servicios y funciones. Los iconos deben ser monocromáticos y alinearse con el color de acento.
- Enfatice el espacio negativo para crear una apariencia limpia y moderna. Utilice un diseño basado en cuadrícula para garantizar un espaciado y una alineación constantes.
- Incorpore animaciones sutiles, como efectos de desplazamiento y transiciones de fundido, para mejorar la participación del usuario sin ser molesto.
